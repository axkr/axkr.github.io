Github Codespaces (BETA)

Just tested Github Codespaces:

  • https://docs.github.com/en/codespaces

In the "PORTS" tab of the terminal, setup port 8080 for "local forwarding"

In the terminal move to the symja_android_library subfolder:

cd symja_android_library

Run the Maven command to create all necessary JARs:

mvn clean install -DskipTests

After that step run one of the Maven commands to start one of the Symja applications

  • https://github.com/axkr/symja_android_library/wiki/Installation

For example:

mvn -f pom.xml exec:java@mmaserver -pl matheclipse-io

Switch to the ports tab and press the web browser icon and append a

/indexmma.html

to the url to see the start page for the mma mopde. Click the "gallery" link or to explore the available functions press the question mark in the upper right corner:
