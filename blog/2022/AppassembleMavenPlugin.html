Appassembler Maven Plugin

Posted by Axel Kramer on April 27, 2022
Tags: blog maven

The appassembler-maven-plugin Maven plugin can be used to create the Symja apps.

Assuming the command will be run from the Maven parent pom.xml file location, it will create a distribution package for the web-interface and the command line apps:

mvn -pl matheclipse-io clean package appassembler:assemble

For Windows the following *.bat files will be generated:
server.bat, mmaserver.bat, symja.bat and mma.bat

You can run one of the following commands:
* bin\server.bat -l -p 8080 to run Symja as a web browser notebook
* bin\mmaserver.bat -l -p 8080 to run Symja as a web browser notebook in Mathematica syntax mode
* bin\symja.bat to run Symja as a console app
* bin\mma.bat to run Symja as a console in Mathematica syntax mode
