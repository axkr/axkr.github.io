Running JBake on Windows

Posted by Axel Kramer on April 27, 2022
Tags: blog maven

To start, let's download JBake and install JBake in a directory C:\jbake-axkr.
Now open a Windows cmd.exe and switch to that folder with the command:

C:\>cd C:\jbake-axkr

You can generate a static site structure with the command

C:\jbake-axkr>bin\jbake.bat -i

Let's generate the static content with the following command in the folder C:\jbake-axkr\output\:

C:\jbake-axkr>bin\jbake.bat -b

Start the embedded server with the following command and open http://localhost:8820/ to control your changes.

C:\jbake-axkr>bin\jbake.bat -s

Create a new page for markdown editing with the *.md file extension, for example in the file

C:\jbake-axkr\jbake-axkr\content\blog\2022\JBakeOnWindows.md

and prepend the meta-data below as the first lines of the file.

title=Running JBake on Windows

date=2022-04-27

type=post

tags=blog

status=published

~~~~~~

Edit the following lines with your blog content for example with the Eclipse Markdown editor and save the page.
Open http://localhost:8820/ again and preview your blog.
If the result is ready for publishing you can commit the content from the folder C:\jbake-axkr\output\ for example as a Github page.
